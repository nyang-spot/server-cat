## 🚀 intro: 서비스 소개
평소에 그냥 지나치는 길고양이들의 매력을 사용자들에게 공유하는 서비스.  
사진을 찍어 귀여운 고양이를 기록하고 우리 동네 인기 고양이를 선정할 수 있습니다.  

<br>

## ⛏️ Skill Set: 기술 스택
- Typescript
- Express, PostgreSQL, Prisma
- Docker, AWS EC2
